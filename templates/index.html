How to format your document

  • Start each question with a number (e.g. 1 What is...)
  • For best image quality: Also upload the same document saved as PDF. In Word, click File → Save As → PDF. The converter will use the PDF to extract pixel-perfect diagrams (with text overlays baked in).
  • MCQ options should be labelled with A B C D
  • Structured questions will be extracted as a whole (including parts/subparts).
  • Indents (e.g., a), (i), etc) should not be used; replace them with plain text instead.
  • If you have answers to the questions, append them at the bottom of the document and begin the answer segment with capitalised text ANSWERS.
  • Answers for parts/subparts should begin with the question number for the first part/subpart but not the subsequent parts/subparts. Refer to sample document question 4's answers — "4" only appears once.
